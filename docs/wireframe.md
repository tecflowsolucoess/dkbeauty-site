# DK Beauty - Wireframe Premium

## Projeto

Nome: DK Beauty

Objetivo:

Criar um site premium para uma maquiadora profissional, transmitindo luxo, sofisticação, confiança e emoção.

O site deverá servir como portfólio da agência e posteriormente ser adaptado para outros segmentos.

---

# Experiência desejada

Quando o visitante acessar o site deverá pensar:

"Essa profissional é extremamente qualificada."

O design deverá transmitir:

• Luxo
• Elegância
• Delicadeza
• Exclusividade
• Confiança
